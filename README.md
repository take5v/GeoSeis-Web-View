# GeoSeis-Web-View

# Installation

Clone repository to local folder

`git clone https://github.com/take5v/GeoSeis-Web-View.git`

Download nodejs from https://nodejs.org.

Then run
`npm install`



# Running

Running with nodejs:
    node bin\www
    
Or nodemon:
    nodemon bin\www
    
Or forever nodejs module:
    forever -w bin\www
